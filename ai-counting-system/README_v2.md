# AI视觉计数系统 - 工业版 v2.0.0

## 功能概述

本系统是专为工业场景设计的AI视觉计数系统，在原有版本基础上进行了全面的工业级优化，具备7×24小时稳定运行能力、超高计数准确率和完善的工业场景适配功能。

## ✨ 核心特性

### 🏭 工业级稳定性
- **7×24小时连续运行**：内存泄漏防护、异常自动重启、看门狗监控
- **断电自动恢复**：计数数据实时持久化，重启后自动恢复到断电前状态
- **异常自修复**：摄像头断开自动重连、帧率过低自动报警、检测异常自动恢复
- **完善日志系统**：运行日志、错误日志、操作日志三级记录，支持导出和问题排查

### 🎯 计数准确率≥99.9%
- **三重防重复计数**：ID跟踪 + 区域防抖 + 时间窗口过滤
- **光照自适应**：自动适应强光、弱光、反光场景，动态调整检测参数
- **运动模糊处理**：高速运动目标模糊校正算法，减少漏检
- **重叠目标优化**：支持轻微重叠、遮挡场景下的准确识别和跟踪

### 🔧 工业场景功能
- **多计数线支持**：最多3条独立虚拟计数线，满足复杂产线需求
- **ROI区域自定义**：支持多边形检测区域，排除无关区域干扰
- **异常报警**：堵料、少件、无料、计数异常自动报警（弹窗、声音、LED）
- **数据持久化**：SQLite数据库存储，支持历史数据回溯和Excel导出
- **多协议对接**：Modbus RTU/TCP、HTTP API、TCP Socket，支持对接PLC、MES系统

### 📷 USB摄像头工业级适配
- **热插拔支持**：摄像头插拔自动检测和重连
- **硬件参数优化**：曝光、增益、白平衡自动适配工业场景
- **帧率稳定性**：丢帧自动补偿，缓存队列平滑帧率波动

### 🖥️ 操作体验优化
- **一键校准**：自动校准计数线位置和检测参数，30秒完成适配
- **权限管理**：操作员/管理员两级权限，防止误操作
- **一键清零**：快速切换批次/班次，操作留痕
- **离线运行**：完全无网络环境下稳定运行

## ⚙️ 主要配置说明

配置文件位于 `config/settings.yaml`，支持通过界面修改，无需手动编辑。

### 计数线配置
支持最多3条计数线，每条可独立设置：
- 位置（0-1，相对于画面高度）
- 计数方向（上/下/双向）
- 启用/禁用状态
- 自定义名称

### 报警配置
- 无料报警：连续30秒无计数触发
- 堵料报警：计数率超过平均值2倍触发
- 低计数率报警：低于10个/分钟触发
- 报警冷却时间：60秒，避免重复报警

### 工业协议
- **Modbus TCP**：默认端口502，支持寄存器读写
- **HTTP API**：默认端口8080，提供RESTful接口
- **TCP Socket**：默认端口9000，支持自定义协议

## 📊 数据存储

系统使用SQLite数据库存储所有数据，支持：
- 计数记录永久保存，可按时间、班次、类别查询
- 操作日志记录所有用户操作，可追溯
- 错误日志记录系统异常，便于排查问题
- 自动备份机制，防止数据丢失

## 🔌 API接口

### HTTP API示例
获取系统状态：
```bash
curl http://localhost:8080/api/status -H "X-API-Key: your_key"
```

获取计数数据：
```bash
curl http://localhost:8080/api/counts
```

重置计数器：
```bash
curl -X POST http://localhost:8080/api/reset -H "Content-Type: application/json" -d '{}'
```
