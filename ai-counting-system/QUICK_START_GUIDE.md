# AI视觉计数系统 v2.0 快速开始指南

## 🚀 5分钟快速配置流程

### 步骤1：启动系统
运行主程序：
```bash
python main_v2.py
```
系统会自动初始化所有模块，创建必要的目录结构。

### 步骤2：打开快速配置向导
点击主界面的「快速配置」按钮（或按快捷键`F12`），启动配置向导。

### 步骤3：配置摄像头
1. 选择要使用的USB摄像头
2. 选择合适的分辨率（推荐1280x720）
3. 调整摄像头角度，确保目标物体在画面中心清晰可见
4. 点击「应用配置」，确认预览画面正常

### 步骤4：拍摄样本照片
按照向导提示拍摄4张照片：
1. 正面照片：物体放在画面中心
2. 侧面照片：物体稍微倾斜
3. 不同位置：物体放在其他位置
4. 不同光照：物体放在光线不同的位置

⚠️ 系统会自动检测照片质量，不合格会提示重拍，确保照片清晰、光照均匀。

### 步骤5：自动训练模型
拍摄完成后，系统会自动：
1. 自动标注样本图片
2. 训练自定义模型
3. 保存模型到`models/custom/`目录
4. 自动激活该产品模型

### 步骤6：开始计数
配置完成后，系统会自动返回主界面，此时：
✅ 摄像头正常工作
✅ 模型已加载完成
✅ 计数线已配置
✅ 可以开始正式计数！

## 常用操作指南

### 快捷键说明
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 开始/暂停计数 |
| `Ctrl+R` | 重置计数器 |
| `F2` | 打开设置界面 |
| `F5` | 抓拍当前画面 |
| `F10` | 导出计数数据 |
| `F12` | 打开快速配置向导 |

### 产品切换
1. 点击「产品管理」→「产品列表」
2. 选择要使用的产品
3. 点击「激活」按钮
4. 系统会自动加载对应的模型和配置
5. 切换完成后即可开始计数

### 数据导出
1. 点击「数据管理」→「导出数据」
2. 选择导出格式：Excel/CSV/PDF
3. 选择筛选条件：时间范围、批次、产品、班次
4. 点击「导出」，文件会保存到`data/exports/`目录

### 批次管理
1. 点击「批次管理」→「新建批次」
2. 输入批次名称，选择关联产品
3. 点击「确定」开始新批次计数
4. 批次结束后点击「完成批次」
5. 可按批次查询和导出统计数据

## MES系统对接

### 启用API服务
1. 打开`config/settings.yaml`文件
2. 找到`mes_api`配置段
3. 设置`enabled: true`
4. 配置`api_key`为您的密钥
5. 重启系统后API服务自动启动

### 接口调用示例
```python
import requests

API_URL = "http://<设备IP>:8000/api/v1"
API_KEY = "your_api_key_here"

headers = {
    "X-API-Key": API_KEY,
    "Content-Type": "application/json"
}

# 获取系统状态
response = requests.get(f"{API_URL}/status", headers=headers)
print(response.json())

# 获取计数统计
response = requests.get(f"{API_URL}/counts", headers=headers)
print(response.json())

# 重置计数器
response = requests.post(f"{API_URL}/reset", headers=headers, json={})
print(response.json())
```

### 主动推送配置
1. 在配置文件中设置`push_enabled: true`
2. 设置`push_url`为MES系统的接收地址
3. 配置`push_interval`为推送间隔（秒，默认1秒）
4. 系统会自动推送计数数据和异常事件到指定地址

## 常见问题

### Q: 计数准确率不高怎么办？
A: 
1. 检查摄像头是否清晰，焦距是否正确
2. 检查光照条件，避免反光或过暗
3. 重新运行快速配置向导，拍摄更多角度的样本
4. 调整检测置信度阈值（默认0.5，可适当调低）

### Q: 出现重复计数怎么办？
A:
1. 检查计数线位置是否合理，避免物体多次跨越
2. 调整防抖帧数（默认5帧，可适当增加）
3. 调整重复计数超时时间（默认1000ms，可适当延长）

### Q: MES接口连接失败怎么办？
A:
1. 检查设备IP地址是否正确
2. 检查端口8000是否开放，防火墙是否允许
3. 检查API密钥是否配置正确
4. 查看日志文件`logs/`目录下的错误信息

### Q: 系统运行卡顿怎么办？
A:
1. 降低摄像头分辨率
2. 降低检测图像尺寸（img_size参数）
3. 关闭不必要的功能（如视频保存、实时预览）
4. 升级硬件配置（推荐使用带GPU的设备）
