# AI视觉智能计数系统 部署文档

## 系统要求
### 最低配置
- CPU: Intel i5-6500 或同等性能
- 内存: 8GB RAM
- 硬盘: 10GB 可用空间
- 操作系统: Windows 10/11 64位

### 推荐配置
- CPU: Intel i7-8700 或同等性能
- 内存: 16GB RAM
- 硬盘: SSD 20GB 可用空间
- 显卡: NVIDIA 独立显卡 (可选，可加速检测)

## 快速部署
### 方式一：直接运行EXE（推荐）
1. 下载发布包 `AI视觉计数系统_v1.0.7z`
2. 解压到任意目录（建议路径不要有中文和空格）
3. 双击运行 `AI视觉计数系统.exe`
4. 程序会自动加载预训练模型，首次运行可能需要10-20秒初始化时间

### 方式二：Python源码运行
1. 安装 Python 3.9 或 3.10 版本（注意：3.11+版本可能存在兼容性问题）
2. 克隆或下载源码到本地
3. 安装依赖:
```bash
pip install -r requirements.txt
```
4. 运行主程序:
```bash
python main.py
```

## 摄像头配置
### 接入USB摄像头
1. 将USB摄像头插入电脑
2. 打开系统设置 -> 摄像头，确认摄像头正常工作
3. 在程序"系统设置"页面中选择对应的摄像头ID（默认0）
4. 点击"开始检测"即可看到实时画面

### 接入网络摄像头/RTSP流
1. 在"系统设置"页面的"默认摄像头"中输入RTSP地址，例如:
```
rtsp://username:password@192.168.1.100:554/stream
```
2. 保存设置后重启程序
3. 点击"开始检测"即可连接网络摄像头

## 模型训练
### 准备训练图片
1. 准备至少4张清晰的目标物体图片
2. 确保图片中目标物体清晰可见，背景不要过于复杂
3. 建议从不同角度、不同光照条件下拍摄
4. 图片格式支持: JPG, PNG, BMP

### 训练步骤
1. 进入"模型训练"页面
2. 点击"添加图片"，选择准备好的4张以上图片
3. 在"类别名称"输入框中输入要识别的物体名称，例如："产品"、"纸箱"、"人员"
4. 点击"开始训练"按钮，等待训练完成（预计2-5分钟）
5. 训练完成后，在下方"已训练模型"列表中找到新训练的模型
6. 双击模型名称即可加载使用

### 训练注意事项
- 确保图片中的目标物体大小适中，不要过小或过大
- 尽量保证图片光照均匀，避免过曝或过暗
- 如果自动标注效果不佳，可以手动标注后再训练
- 训练过程中不要关闭程序，否则会训练失败

## 计数线设置
1. 在主界面拖动"计数线位置"滑块可以调整计数线的上下位置
2. 计数线默认在视频画面中间位置（50%）
3. 建议将计数线设置在目标物体经过的中间位置
4. 可以根据实际场景灵活调整计数线高度

## 方向设置
1. 进入"系统设置"页面
2. 在"计数方向"下拉框中选择:
   - both: 双向计数（默认）
   - up: 只统计向上移动的目标
   - down: 只统计向下移动的目标
3. 保存设置后生效

## 数据导出
1. 进入"数据报表"页面
2. 选择要导出的日期范围
3. 点击"查询"按钮查看数据
4. 点击"导出Excel"或"导出CSV"按钮
5. 选择保存路径即可导出数据

## LED看板对接
### 硬件连接
1. 将LED看板通过RS232/RS485串口连接到电脑
2. 确认串口驱动已正确安装
3. 在"设备管理器"中查看串口号（例如COM1, COM2等）

### 软件配置
1. 进入"系统设置"页面
2. 勾选"启用LED对接"
3. 输入正确的串口号和波特率（默认9600）
4. 保存设置后重启程序
5. 程序会自动将实时计数数据发送到LED看板

### 通信协议
默认通信协议（可根据实际LED看板修改）:
- 波特率: 9600
- 数据位: 8
- 停止位: 1
- 校验位: 无
- 发送格式: `T{总计数},U{向上计数},D{向下计数}\r\n`
- 示例: `T1234,U567,D667\r\n`

## 性能优化
### 提升帧率
1. 在"系统设置"中降低"置信度阈值"到0.4
2. 降低视频分辨率到640x480
3. 关闭不必要的后台程序
4. 如果有NVIDIA显卡，可以将"运行设备"设置为cuda

### 提高准确率
1. 提高"置信度阈值"到0.6以上
2. 增加"防抖帧数"到5-10
3. 调整计数线位置，避免目标在计数线附近徘徊
4. 确保视频画面清晰，没有抖动

## 常见问题
### Q: 程序打开后闪退怎么办？
A: 
1. 检查路径是否有中文或特殊字符
2. 确认已安装Visual C++ Redistributable运行库
3. 尝试以管理员身份运行程序

### Q: 检测帧率低怎么办？
A:
1. 降低视频分辨率
2. 关闭其他占用CPU的程序
3. 尝试使用独立显卡运行
4. 减少画面中同时检测的目标数量

### Q: 计数不准确怎么办？
A:
1. 调整置信度阈值，过滤误检测
2. 增加防抖帧数，避免重复计数
3. 调整计数线位置
4. 检查视频是否有抖动，固定摄像头

### Q: 训练模型失败怎么办？
A:
1. 确认图片数量至少4张
2. 检查图片是否损坏
3. 确保磁盘有足够的剩余空间
4. 尝试重启程序后重新训练

### Q: 如何升级程序？
A:
1. 备份 `models/custom` 目录下的自定义模型
2. 备份 `data` 目录下的历史数据
3. 下载新版本发布包，覆盖原有文件
4. 将备份的模型和数据复制回对应目录
